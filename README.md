# Srvey

Srvey is a modern survey platform designed to make the survey experience more engaging, interactive, and enjoyable for participants while giving creators powerful tools to collect structured insights.

## Overview
Srvey transforms traditional boring surveys into interactive experiences.
Instead of static question forms, Srvey introduces dynamic flows, progress feedback, gamified elements, and smart question logic to improve engagement and completion rates.

The platform supports secure role-based access, survey management, and response analytics.

## Key Goals
- Improve participant engagement and completion rates
- Provide dynamic and interactive survey flows
- Support secure authentication & role-based access
- Enable structured survey creation and response tracking
- Deliver meaningful analytics for survey creators

## Tech Stack
Frontend: Next.js
Backend: FastAPI (Python)
Database: MongoDB
